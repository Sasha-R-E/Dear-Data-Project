# Commute Visualization Sketch

This project visualizes my commute to and from university across **100 different days**.  
Each color represents the mode of transport used during that day.

## Features
- Visual representation of daily commutes
- Color-coded transport modes
- Interactive scrolling and zooming

## How to Interact
- **Scroll:** Click and drag the scrollbar, then release to move through the days  
- **Zoom in:** Press the **RIGHT arrow key**  
- **Zoom out:** Press the **LEFT arrow key**  
- **Exit:** Click anywhere within the sketch, then press the **Esc** key  

## Notes
- Designed for exploration and pattern recognition in commuting habits  
- Built as a creative sketch to combine data visualization with interactivity
